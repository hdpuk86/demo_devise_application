# Rails 6 Devise Demo Application

### Build Features:
- postgresql
- webpack
- bootstrap
- Users Controller (index, edit, update, show)

## Getting Started
### Requirements
- Ruby 2.5.1
- Rails ~> 6.0.2 >= 6.0.2.2

### Setup
`bundle`  

`yarn`  

`rails db:create`  

`rails db:migrate`  

`rails server`
